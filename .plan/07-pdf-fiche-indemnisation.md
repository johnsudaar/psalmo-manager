# 07 — PDF Fiche d'indemnisation animateur

## Overview

The "Fiche d'indemnisation animateur" is a one-page (sometimes two-page) PDF document generated
for each staff member. It summarises all financial flows between the staff member and the
Psalmodia association for a given edition.

- **Triggered by**: `GET /staff_profiles/:id/fiche.pdf`
- **Generated by**: `app/pdf/fiche_indemnisation_pdf.rb` (Prawn document class)
- **Streamed directly** to the browser — no file storage
- **Layout style**: custom clean layout (not a pixel-perfect replica of the Google Sheet)

---

## Page Layout

### Page size and margins

- **Page size**: A4 (portrait)
- **Margins**: top: 20mm, bottom: 20mm, left: 20mm, right: 20mm
- **Font**: Helvetica (built into Prawn, no font loading needed)
- **Primary colour**: `#4338CA` (Indigo 700) for section headers
- **Body text**: `#111827` (gray-900)
- **Light rule colour**: `#E5E7EB` (gray-200)

### 5. Montant dû à l'animateur

Large highlighted box (light indigo background `#EEF2FF`, border `#4338CA`):

```
Montant dû à l'animateur :    490,60 €
```

Bold, size 12. This is `amount_owed_to_instructor_cents`.

Formula reminder (see `.plan/10-business-rules.md`):
```
amount_owed = total_to_pay + total_psalmodia_covers - total_member_uncovered + total_member_covered
```

|---|---|---|

---

### 8. Soldes finaux

Two highlighted rows side by side or stacked:

```
Somme à payer à l'animateur :    [amount or "—"]
Somme à payer à Psalmodia :      [amount or "—"]
```

Logic:
- If `balance_cents > 0`: "Somme à payer à l'animateur" = `balance_cents`, "Somme à payer à Psalmodia" = `—`
- If `balance_cents < 0`: "Somme à payer à Psalmodia" = `balance_cents.abs`, "Somme à payer à l'animateur" = `—`
- If `balance_cents == 0`: both = `0,00 €`

The non-zero row uses a stronger background colour to draw attention.

---

### 9. Footer

Thin rule, then:
```
Document généré le {Date.today.strftime("%d/%m/%Y")} — Psalmodia {year}
```
Right-aligned, size 8, gray.

## Prawn Implementation Guide

### File location

`app/pdf/fiche_indemnisation_pdf.rb`

### Class structure

```ruby
class FicheIndemnisationPdf
  include Prawn::View

  def initialize(staff_profile)
    @sp      = staff_profile
    @person  = staff_profile.person
    @edition = staff_profile.edition
    @document = Prawn::Document.new(
      page_size:   "A4",
      page_layout: :portrait,
      margin:      [20.mm, 20.mm, 20.mm, 20.mm]
    )
  end

  def build
    header
    section_frais_animateur
    section_frais_pris_en_charge
    section_frais_membres
    section_montant_du
    section_acomptes
    section_versements
    section_soldes
    footer
    self
  end

  def render
    build
    document.render
  end

  private

  INDIGO    = "4338CA"
  INDIGO_BG = "EEF2FF"
  GRAY      = "6B7280"
  RULE      = "E5E7EB"

  def section_header(text)
    fill_color INDIGO
    fill_rectangle [0, cursor], bounds.width, 16
    fill_color "FFFFFF"
    text_box text, at: [4, cursor - 3], size: 8, style: :bold
    fill_color "111827"
    move_down 20
  end

  def two_col_row(label, amount_cents, bold: false)
    style = bold ? :bold : :normal
    formatted = format_euros(amount_cents)
    table([[label, formatted]], width: bounds.width,
          cell_style: { borders: [], size: 9, font_style: style })
  end

  def format_euros(cents)
    return "—" if cents.nil?
    "#{"%.2f" % (cents / 100.0)}".gsub(".", ",") + " €"
  end
end
```

### Rendering in controller

```ruby
# app/controllers/staff_profiles_controller.rb
def fiche
  @staff_profile = StaffProfile
    .includes(:person, :edition, :staff_advances, :staff_payments)
    .find(params[:id])

  pdf = FicheIndemnisationPdf.new(@staff_profile).render

  send_data pdf,
            filename:    "fiche_#{@staff_profile.dossier_number}_#{@staff_profile.person.last_name}.pdf",
            type:        "application/pdf",
            disposition: "inline"
end
```

## Edge Cases

| Scenario | Handling |
|---|---|
| No advances | Show "Aucun acompte enregistré", Total = 0,00 € |
| No payments | Show "Aucun versement enregistré", Total = 0,00 € |
| All amounts are zero | Show 0,00 € everywhere, balance = 0,00 € |
| Long name overflows | Truncate to 40 chars in header, full name in body |
| Balance exactly zero | Both "Somme à payer" rows = 0,00 €, no highlighting |
| km_rate is overridden | Show rate with note "(taux personnalisé)" |
| No transport (km = 0) | Frais de déplacement = 0,00 €, no sub-label |
| Content exceeds one page | Prawn handles page breaks automatically |

## mm Helper

Prawn uses points (1pt = 1/72 inch). Add a helper for millimetre conversion:

```ruby
# In the PDF class or a shared module
def mm(val)
  val * 2.8346  # 1mm = 2.8346pt
end
```

Or use `Prawn::Measurements` which provides `mm2pt`.
